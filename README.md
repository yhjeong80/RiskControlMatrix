# RCM Login / Role Demo with Risk Rating Calculation

추가 반영 사항
- 발생 가능성 / 결과의 심각성 값은 1~5 드롭다운 선택
- 고유 Risk Rating 자동 계산
- 잔여 Risk Rating 자동 계산
- Rating 값은 색상 배지로 표시
  - Low: 초록
  - Medium: 노랑
  - High: 빨강

수식 로직
- 1~6: Low
- 7~14: Medium
- 15~25: High

예시
- 발생 가능성 3, 결과의 심각성 2 → 6 → Low
